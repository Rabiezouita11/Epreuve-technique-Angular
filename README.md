# Epreuve-technique-Angular
Epreuve-technique-Angular web application using Angular 15  | Node.js | Express.js | mongoose
## Steps for Project Execution :


##### Docker Install
```
docker-compose up
```


##### Client Install
```
cd ziston
npm install 
ng serve 
```

##### Server Install
```
cd Backend_Ziston
npm install 
npm install nodemon
npm run dev 
```
